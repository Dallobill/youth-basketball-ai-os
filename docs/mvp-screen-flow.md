# MVP Screen Flow

## 1. Dashboard
Purpose:
- show active teams
- show latest practice/game entries
- show players needing review
- show recent AI reports

Widgets:
- active team cards
- upcoming sessions
- player priority list
- recent summary outputs

## 2. Team Page
Purpose:
- view roster
- view recent practices/games
- start new evaluation
- generate team summary

Sections:
- roster
- attendance trend
- last 5 practices
- last 5 games
- latest team issues

## 3. Player Page
Purpose:
- central development record for one athlete

Sections:
- profile header
- latest evaluation scores
- development priorities
- attendance log
- clip notes
- generated reports

## 4. Evaluation Form
Purpose:
- rapid coach entry after practice or game

Fields:
- player
- evaluation type
- rating sliders or number inputs
- strengths
- priorities
- notes

Design rule:
- must be completable in under 2 minutes per player

## 5. Practice Entry
Purpose:
- log objective, notes, and team themes

Fields:
- date
- objective
- strengths
- issues
- next focus

## 6. AI Output View
Purpose:
- review generated reports before saving or sharing

Tabs:
- player summary
- team summary
- practice plan
- parent update
